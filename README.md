## 🎂 Chaewon-Birthday
친구의 생일을 축하하기 위해 트위터(X)의 UI를 웹으로 재현하고, 

맞춤형 콘텐츠로 구성한 이벤트 페이지 프로젝트입니다.




### 📖 Project Overview

컨셉: 트위터(X) 인터페이스를 활용한 디지털 생일 편지 및 추억 아카이빙

제작 기간: 2025.11 (생일 기념)

주요 기능: - X(Twitter) 다크모드 UI 클론 코딩

커스텀 트윗 피드 (친구 맞춤형 텍스트 및 이미지)

반응형 레이아웃 (Grid & Flexbox 활용)

### 🛠 Tech Stack
Frontend: HTML5, CSS3

Design: X(Twitter) UI Design System Reference

Collaboration Tool: ChatGPT (코드 구조 설계 및 레이아웃 최적화 도움)


### 🤖 AI Collaboration Experience
이 프로젝트는 AI와 협업하여 효율적으로 완성되었습니다.

레이아웃 설계: 복잡한 3열 구조의 반응형 레이아웃을 잡는 데 AI의 가이드를 활용했습니다.

CSS 최적화: 이미지 그리드(object-fit: cover) 및 둥근 모서리 처리 등 디테일한 스타일링 코드를 AI와 검토하며 수정했습니다.

문제 해결: 단일 이미지와 다중 이미지의 높이 불균형 문제를 AI와 논의하여 안정적인 피드 구조를 완성했습니다.
